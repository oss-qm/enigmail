Informação de actualização para o Enigmail 0.8x

O Enigmail v0.8x para o Thunderbird recebeu uma nova interface de utilizador! Se estava familiarizado com o Enigmail de versões anteriores a 0.81.0 leia, por favor, esta informação antes de prosseguir.

Configurar o Enigmail

A configuração do Enigmail está agora dividida em duas partes:

  • Opções específicas de Conta para cada Identificador de chave PGP, configuração por omissão para assinar / cifrar
  • Definições globais comuns

Opções específicas de Conta

As opções específicas de conta estão disponíveis através do menu Ferramentas -> Opções de Conta. Para cada conta há agora um item Segurança OpenPGP em que se pode configurar as opções específicas de cada conta.

As configurações de chave PGP do Enigmail v.0.80.0 e anteriores não são convertidas para as novas opções por conta. O Enigmail está desactivado para todas as contas!

Preferências Globais Comuns

As preferências globais comuns foram arrumadas num único diálogo. As preferências estão disponíveis quer a partir do menu Enigmail, quer clicando o botão Avançado nas Preferências de Segurança OpenPGP das Opções de Conta.

Foi adicionada uma nova opção que lhe permite activar automaticamente a encriptação ao responder a mensagens cifradas. Note que esta opção não influencia a configuração da assinatura da mensagem.

Compor Mensagens

O comportamento do Enigmail foi adaptado para se assemelhar mais ao comportamento do S/MIME. Foi adicionado um novo botão OpenPGP a partir do qual se podem alterar as opções de assinar / cifrar. O menu Enigmail já não é usado para enviar mensagens assinadas ou cifradas; o envio é efectuado apenas pelo botão Enviar ou o menu Ficheiro -> Enviar. Sugere-se activamente que teste as novas opções enviando algumas mensagens para si próprio antes de usar o Enigmail efectivamente, para que perceba o seu novo comportamento.

As opções para assinar ou cifrar são agora visíveis no canto inferior direito da janela de composição, através de pequenos ícones mostrando se a assinatura ou encriptação estão activas.

Problemas conhecidos

  • O Enigmail ainda não consegue memorizar senhas diferentes para chaves OpenPGP diferentes. Pode introduzir a senha de cada vez que for necessário ou usar a mesma senha para todas as suas chaves.
