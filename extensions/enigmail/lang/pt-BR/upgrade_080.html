Informações de Atualização para o Enigmail 0.8x

O Enigmail v0.8x para Thunderbird recebeu uma nova interface de usuãrio! Se você está familiarizado com as versões do Enigmail anteriores a 0.81.0, por favor leia estas informações antes de continuar.

Configurando o Enigmail

A configuração do Enigmail agora é dividida em duas partes:

  • Configurações específicas de conta para o ID de chave PGP, criptografia / assinatura padrões
  • Preferências Globais Comuns

Configurações Específicas de Conta

As configurações específicas de conta estão disponíveis através do menu Ferramentas -> Configurações de Conta. Para cada conta agora existe um item na lista Segurança OpenPGP que contém as opções específicas para cada conta.

As configurações de chave PGP do Enigmail v0.80.0 e versões mais antigas não são convertidas no novo formato por conta. O Enigmail é por padrão desabilitado para todas as contas!

Preferências Globais Comuns

As configurações globais comuns foram re-organizadas e combinadas em um diálogo único. As preferências estão disponíveis através do menu do Enigmail, ou clicando no botão Avançado nas Preferências de Segurança OpenPGP do Gerenciador de Contas.

Uma nova opção foi acrescentada, que permite que você ligue a criptografia automaticamente se estiver respondendo a uma mensagem criptografada. Note que isto não influencia as configurações de assinatura de mensagem.

Compondo Mensagens

O comportamento do Enigmail foi adaptado para ser mais semelhante ao comportamento do S/MIME. Um novo botão OpenPGP foi acrescentado, a partir do qual as configurações de criptografia / assinatura podem ser selecionadas. O menu do Enigmail não serve mais para enviar mensagens criptografadas ou assinadas, todo o envio agora é feito a partir do botão Enviar ou através do menu Arquivo -> Enviar. É recomendado que voê tente as novas configurações enviando algumas mensagens para si mesmo antes de utilizá-lo ativamente, de forma que você se entenda o funcionamento do novo comportamento do Enigmail.

As configurações para criptografar e assinar uma mensagem agora estão visíveis no canto inferior direito da janela de composição de mensagem, com pequenos ícones sendo mostrados caso a assinatura ou criptografia estejam ligadas.

Problemas Conhecidos

  • O Enigmail ainda não é capaz de lembrar de múltiplas senhas para chaves PGP diferentes. É necessário digitar a senha todas as vezes, ou configurar a mesma senha para todas as chaves que você estiver utilizando.
