<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd"> <html xmlns="http://www.w3.org/1999/xhtml"> <head> <title>Axuda de Enigmail: Como resolver problemas ao inicializar OpenPGP</title> <style type="text/css"> /*<![CDATA[*/ dt { font-weight: bold; } /*]]>*/ </style> </head> <body> <h1>Axuda de Enigmail</h1> <h3>Como resolver problemas ao inicializar OpenPGP</h3> <p>Hai varias razóns polas que OpenPGP pode fallar. As máis comúns descríbense a continuación; para máis información visite a<a href="http://enigmail.mozdev.org/support/" target="_blank">páxina de soporte de Enigmail.</a></p> <dl> <dt>Non se atopou GnuPG</dt> <dd> <p> Para que funcione OpenPGP, ten que estar instalada a ferramenta <a href="http://www.gnupg.org/" target="_blank">GnuPG</a>. Se non se atopa GnuPG, o primeiro que ten que facer é asegurarse de que o executábel <tt>gpg.exe</tt> (en Windows; <tt>gpg</tt> noutras plataformas) estea instalado no seu computador. Se GnuPG está instalado, e OpenPGP non o atopa, entón é preciso definir manualmente a ruta a GnuPG en Preferencias de OpenPGP (menu <i>OpenPGP</i> &gt; <i>Preferencias</i>) </p> <dt>Enigmime falla ao inicio</dt> <dd> <p> OpenPGP só traballa se foi compilado utilizado a mesma contorna de compilación que Thunderbird ou SeaMonkey. Isto significa que pode utilizar as versións oficiais de Enigmail só se utiliza as versións de Thunderbird ou SeaMonkey proporcionadas por <a href="www.mozilla.org">mozilla.org</a>. </p> <p> Se quere usar unha versión de Thunderbird ou SeaMonkey que ven doutra fonte (p.e. o provedor da súa distribución de Linux), ou se quere compilar a aplicación por si mesmo/a, debería utilizar unha versión de Enigmail compilada pola mesma fonte, ou compilada por si mesmo/a. Para compilar Enigmail, consulte a <a href="http://enigmail.mozdev.org/download/source.php">sección de Código Fonte</a> na páxina web de Enigmail. Non cree ningún informe de erros referente a este problema, non ten solución. </p> </dd> <p>Pode obter máis axuda na <a href="http://enigmail.mozdev.org/support/troubles.php" target="_blank">Páxina web de soporte de Enigmail</a>.</p> </body> </html>
