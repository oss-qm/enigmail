Información de Actualización para Enigmail 0.8x

Enigmail v0.8x para Thunderbird ha recibido una nueva interfaz de usuario! Si usted está familiarizado con las versiones de Enigmail previas a 0.8.1.0, por favor lea esta información antes de proceder.

Configurando Enigmail

La configuración de Enigmail ahora está separada en dos partes:

  • Configuración específica de Cuenta para PGP key ID, cifrado / firma por defecto
  • Preferencias globales comunes

Configuración Específica de Cuenta

La configuración específica de cuenta está disponible mediante el menú Herramientas -> Configuración de Cuentas. Para cada cuenta ahora hay un item de lista Seguridad OpenPGP que contiene las opciones específicas para cada cuenta..

La configuración de claves PGP de Enigmail v0.80.0 y versiones anteriores no son convertidas a la nueva configuración por cuentas. Por defecto, Enigmail está deshabilitado para todas las cuentas!

Preferencias Comunes Globales

Las preferencias comunes globales han sido limpiadas y fundidas en un diálogo. Las preferencias están disponibles ya sea mediante el menú Enigmail, o clickeando el botón Avanzadas en las Preferencias de Seguridad de OpenPGP en la Configuración de Cuentas.

Una nueva opción ha sido añadida que permite activar el cifrado automáticamente si se responde a un mensaje cifrado. Note que esto no influye la configuración para firmado de mensajes.

Redactando Mensajes

El comportamiento de Enigmail ha sido adaptado para ser más similar al comportamiento de S/MIME. Ha sido agregado un nuevo botón OpenPGP, desde el cual se puede seleccionar la configuración de cifrado / firma. El menú Enigmail ya no es utilizado para enviar mensajes cifrados o firmados, el envio se realiza únicamente desde el botón normal Enviar o desde el menú Archivo -> Enviar. Se sugiere que usted pruebe estas nuevas configuraciones enviándose un par de mensajes a sí mismo antes de usarlo actívamente, de modo que usted comprenda el nuevo comportamiento.

La configuración para cifrar o firmar un mensaje ahora es visible en la esquina inferior derecha de la ventana de redacción, con pequeños íconos mostrando si está activado el cifrado o la firma..

Problemas Conocidos

  • Enigmail aún no es capáz de recordar varias contraseñas para diferentes claves PGP. O bien escriba la contraseña por cada vez, o utilice la misma contraseña para todas las claves que esté utilizando.
