Πληροφορίες αναβάθμισης για το Enigmail 0.8x

Το Enigmail v0.8x για τον Thunderbird έχει νέο γραφικό περιβάλλον! Αν έχετε χρησιμοποιήσει εκδόσεις του Enigmail προγενέστερες της 0.81.0, διαβάστε τις παρακάτω πληροφορίες πριν συνεχίσετε.

Ρύθμιση του Enigmail

Οι ρυθμίσεις του Enigmail έχουν χωριστεί σε δύο μέρη:

  • Ρυθμίσεις ανά λογαριασμό για ID κλειδιών PGP key ID, προκαθορισμένη χρήση κρυπτογράφησης / υπογραφής
  • Κοινές - καθολικές ρυθμίσεις

Ρυθμίσεις ανά λογαριασμό

Οι ρυθμίσεις ανά λογαριασμό είναι διαθέσιμες μέσω του μενού Επεξεργασία -> Ρυθμίσεις λογαριασμού. Για κάθε λογαριασμό υπάρχει η καρτέλα Ασφάλεια OpenPGP όπου ρυθμίζονται οι παράμετροι του OpenPGP.

Οι ρυθμίσεις κλειδιών PGP του Enigmail v0.80.0 και προγενέστερων εκδόσεων δε μετατρέπονται αυτόματα στο νέο σχήμα. Το Enigmail είναι αρχικά απενεργοποιημένο για όλους τους λογαριασμούς!

Κοινές - καθολικές ρυθμίσεις

Οι κοινές - καθολικές ρυθμίσεις έχουν περιληφθεί σε ένα διάλογο. Οι προτιμήσεις είναι διαθέσιμες μέσω του μενού του Enigmail, ή κάνοντας κλικ στο κουμπί Για προχωρημένους Προτιμήσεις Ασφαλείας του Διαχειριστή λογαριασμών.

Έχει προστεθεί μία νέα επιλογή που ενεργοποιεί την αυτόματη κρυπτογράφηση όταν απαντάτε σε ένα κρυπτογραφημένο μήνυμα. Σημειώστε ότι αυτό δεν επηρεάζει τις ρυθμίσεις υπογραφής μηνυμάτων.

Σύνταξη μηνυμάτων

Η συμπεριφορά του Enigmail έχει αλλάξει και μοιάζει πλέον με αυτή του S/MIME. Έχει προστεθεί ένα νέο κουμπί OpenPGP από το οποίο μπορούν να ρυθμιστούν οι προτιμήσεις για την κρυπτογράφηση / υπογραφή. Η αποστολή κρυπτογραφημένων ή υπογεγραμμένων μηνυμάτων δε γίνεται πια από το μενού του Enigmail, αλλά από το κουμπί Αποστολή ή από το μενού Αρχείο -> Αποστολή. Σας προτείνεται να δοκιμάσετε τις νέες σας ρυθμίσεις στέλνοντας μερικά μηνύματα στον εαυτό σας πριν αρχίσετε να το χρησιμοποιείτε κανονικά, ώστε να καταλάβετε τη νέα λειτουργία του.

Οι ρυθμίσεις για την κρυπτογράφηση ή υπογραφή του μηνύματος είναι ορατές στο κάτω δεξιό τμήμα του παραθύρου Σύνταξης, όπου εμφανίζονται μικρά εικονίδια που δηλώνουν την τρέχουσα κατάσταση.

Γνωστά προβλήματα

  • Το Enigmail δεν έχει προς το παρόν τη δυνατότητα απομνημόνευσης ξεχωριστής φράσης πρόσβασης για κάθε κλειδί PGP. Έτσι θα πρέπει να το πληκτρολογείτε κάθε φορά ή να ορίσετε την ίδια φράση πρόσβασης για κάθε κλειδί που χρησιμοποιείτε.
