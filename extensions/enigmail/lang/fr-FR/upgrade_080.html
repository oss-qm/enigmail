Informations de mise à jour pour Enigmail 0.8x

Enigmail v0.8x pour Thunderbird comporte une nouvelle interface utilisateur ! Si vous êtes habitués avec une version d'Enigmail antérieures à la 0.81.0, veuillez lire ces informations avant de continuer.

Configurer Enigmail

La configuration d'Enigmail est maintenant divisée en deux parties :

  • Réglages spécifiques au compte pour l'identifiant de clef, le chiffrement/signature par défaut
  • Préférences globales communes

Réglages spécifiques au compte

Les réglages spécifiques au compte sont disponibles dans le menu Outils -> Paramètres des comptes. Chaque compte possède une entrée Sécurité OpenPGP qui contient les options spécifiques pour chaque comptes.

Les paramètres de clef PGP pour les version d'Enigmail inférieure ou égale à 0.80.0 ne sont pas converties vers les nouveaux paramètres spécifiques à chaque compte. Enigmail est, par défaut, désactivé pour tous les comptes !

Préférences globales communes

Les préférences globales ont été nettoyées et fusionnées dans une boite de dialogue. Les préférences sont accessibles soit depuis le menu Enigmail soit en cliquant le bouton Avancées dans le panneau Sécurité OpenPGP du gestionnaire des comptes.

Une nouvelle option est apparue permettant d'activer automatiquement le chiffrement lors de la réponse à un message chiffré. À noter que ceci n'influe pas sur les réglages de signature des messages.

Rédaction de messages

Le comportement d'Enigmail a été adapté pour être plus semblable au comportement de S/MIME. Un nouveau bouton OpenPGP a été ajouté permettant de sélectionner les paramètres de chiffrement/signature. Le menu Enigmail ne sert plus à l'envoi de messages chiffrés ou signés, l'envoi se fait uniquement via l'habituel bouton Envoyer ou depuis le menu Fichier -> Envoyer. Il est recommandé de tester les nouveaux réglages en vous envoyant quelques messages à vous-même avant de l'utiliser de manière intense, de manière à bien comprendre le nouveau fonctionnement.

Les réglages pour le chiffrement ou la signature sont dorénavant visibles dans le coin inférieur droit de la fenêtre de composition, sous la forme de petites icônes affichées si la signature ou le chiffrement sont activés.

Problèmes connus

  • Enigmail n'est pour l'instant pas capable de retenir plusieurs mot de passe pour des clefs PGP différentes. Il faut saisir le mot de passe à chaque fois, ou choisir le même mot de passe pour toutes les clefs utilisées.
