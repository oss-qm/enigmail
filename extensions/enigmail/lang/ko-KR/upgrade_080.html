Enigmail 0.8x 업그레이드 정보

Enigmail v0.8x for Thunderbird가 새로운 인터페이스를 가지게 되었습니다! 0.81.0 이전의 Enigmail 버전에 익숙하신 분이라면, 계속하시기 전에 다음 정보를 읽어주시기 바랍니다.

Enigmail 설정

Enigmail 설정은 이제 두 부분으로 나누어집니다:

  • PGP 키 ID, 기본 암호화 / 서명에 대한 계정별 설정
  • 일반 전역 설정

계정별 설정

도구 → 계정... 메뉴를 통하여 계정별 설정이 가능합니다. 각 계정마다 계정에 한정된 설정을 할 수 있는 OpenPGP 보안 목록 항목이 있습니다.

Enigmail v0.80.0 을 포함한 이전버전의 PGP 키 설정은 새로운 계정별 설정으로 변환되지 않습니다. Enigmail은 모든 계정에 대해 기본으로 비활성 상태로 되어있습니다!

일반 전역 설정

일반 전역 설정은 간결화 후 하나의 대화상자로 통합되었습니다. 설정은 Enigmail 메뉴나 계정 메뉴의 OpenPGP 보안 설정에서 고급 버튼을 누르면 사용가능합니다.

암호화된 메시지에 답장을 쓸 경우 자동적으로 암호화를 하도록하는 새로운 옵션이 추가되었습니다. 메시지 서명에 대한 설정에는 영향을 미치지 않음에 주의하시기 바랍니다.

메시지 작성하기

Enigmail의 동작 방식은 S/MIME 동작 방식에 좀 더 유사하도록 만들어졌습니다. 암호화 / 서명 설정에서 선택할 수 있는 새로운 OpenPGP 버튼이 추가되었습니다. Enigmail 메뉴가 암호화 또는 서명한 메일을 보내기를 수행하지 않고, 기본 보내기 버튼 혹은 파일 → 보내기 메뉴에서만 보낼 수 있게 되었습니다. 본격적으로 사용하기 전에 자신에게 메일 몇 통을 시험적으로 보내서 새로운 설정이 어떻게 작동하는지 시험해보시기를 권해드립니다.

메시지에 암호화 또는 서명하기 설정은 작성 윈도우의 오른쪽 코너에 서명 또는 암호화가 되었는지를 나타내는 작은 아이콘으로 나타나는 것을 보실 수 있습니다.

알려진 문제점

  • Enigmail은 아직 서로 다른 PGP 키에 대하여 여러 개의 암호를 기억하지 못합니다. 각각에 대하여 매번 암호를 입력하든지, 사용하는 키에 대하여 같은 암호를 사용해 주시기 바랍니다.
